# ASCII Art Generator

A free online tool to generate ASCII art from text. Create cool text banners for social media, code comments, and more.

## Features

- Multiple ASCII art fonts and styles
- Real-time preview
- Customizable width and character set
- Support for special characters
- Copy or download as text file
- 20+ font styles including Block, Slant, Shadow, and more

## How to Use

1. Enter your text in the input field
2. Select a font style
3. Adjust settings (width, characters)
4. Copy or download the generated ASCII art

## Popular Fonts

- **Block** — Classic bold block letters
- **Slant** — Italic-style slanted text
- **Shadow** — Letters with shadow effect
- **Banner** — Simple banner style
- **Star Wars** — Star Wars themed font
